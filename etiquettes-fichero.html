Imprimante FICHERO Bluetooth — via Web Bluetooth (nécessite Bluefy sur iPhone)
À propos

Cette page parle directement en Bluetooth à l'imprimante FICHERO (protocole documenté par le projet open-source fichero-printer, licence MIT), sans passer par l'application FICHERO Printer. Les patients enregistrés sont stockés uniquement dans ce téléphone (mémoire du navigateur), jamais envoyés sur internet. Safari ne supporte pas le Bluetooth depuis une page web : cette page doit être ouverte avec l'appli Bluefy (gratuite, App Store).
